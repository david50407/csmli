# csmli

Mini-Lisp interpreter

__No unit tests yet__

## Compile

```bash
$ crystal compile --release src/csmli.cr
```

## Usage

Append lisp program filename as argument or leave blank for input from standard input.

```bash
$ ./csmli [program.lsp]
```
